What it does

From one oversized model to plated 3MF projects.

Keep the creative scale of the model while automating the repetitive cutting, connector, routing, and plate-layout work.

01

Scale precisely

Set a final height, width, depth, longest side, or scale factor using millimetres, centimetres, inches, or feet.

02

Cut for your beds

Select one or more printers. Pieces are cut and routed according to the available build volumes, including custom printer sizes.

Plan assembly

Add press-fit or glue-fit pegs, recessed matching codes, tiny-piece avoidance, and optional hollowing before export.

Inspect the exploded assembly, then generate 3MF projects with pieces already oriented and arranged across print plates.

Slicer compatibility

Know what opens as a complete project.

Scale & Cut 3D uses Bambu/Orca project metadata to retain automatic plate assignments. Some slicers can open the model geometry without preserving those plates.

SlicerCompatibilityWhat to expect
OrcaSlicerFull project

Opens the generated plates, arranged pieces, and project data.

Bambu StudioFull project

Opens the generated plates, arranged pieces, and project data.

QIDI StudioPlates verified*

Preserves generated plate layouts. Select a compatible QIDI printer, material, and process profile before slicing.

PrusaSlicerGeometry only

Can recover the model pieces, but does not retain Bambu/Orca plate organization or project settings.

CuraGeometry only

May import model geometry, but the automatic plate layout is not supported.

Creality Print 7.1 / ElegooSlicer 1.1.8Not compatible

These versions reject the current plated project format.

*QIDI Studio compatibility was verified with a seven-plate Bambu-format project. Slicer format support can change between versions.

Questions

Before you buy.

What does Scale & Cut 3D do?

It scales oversized 3D models, divides them into pieces that fit selected printer beds, optionally adds alignment connectors, and exports slicer-ready projects.

Which slicers are supported?

OrcaSlicer and Bambu Studio have full plated-project support. QIDI Studio has been verified to preserve generated plate layouts after you select the correct QIDI profiles. PrusaSlicer and Cura are geometry-only options and do not preserve the automatic plates. Creality Print 7.1 and ElegooSlicer 1.1.8 are not compatible with the current project format.

Which model formats can it open?

STL, OBJ, 3MF, PLY, and OFF files are supported.

Does one license cover Windows and Mac?

Yes. One purchase includes both downloads and activation on up to two computers. The Mac build currently requires Apple silicon.
